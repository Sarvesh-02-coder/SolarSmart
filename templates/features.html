🔆 Project Features

Explore what makes SolarSmart intelligent and user-friendly!

⚡ Live Weather-Driven Power Prediction

Using NASA POWER and OpenStreetMap APIs, we fetch weather data dynamically based on user pincode.

🔍 Panel Technology Customization

Supports 15+ modern solar technologies and manufacturers for accurate results.

📈 Daily, Monthly, and Yearly Analytics

Clear insights into current, forecasted, and historical energy generation in kWh.

💰 Savings Estimation

Calculates potential money saved annually based on electricity tariffs.

📊 Charts and Visualization

Clean, dynamic charts rendered using Chart.js to visualize solar output trends.
